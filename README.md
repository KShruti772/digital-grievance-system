# Digital Grievance System

## 📌 Overview
A web-based platform that allows citizens to register complaints and track grievance status efficiently.

## 🚀 Features
- User Authentication (Login/Register)
- Complaint Registration
- Status Tracking
- Admin Dashboard
- Location-based complaint tracking

## 🛠 Tech Stack
- Frontend: HTML, CSS, JavaScript
- Backend: Flask / Python
- Database: SQLite / MySQL

## ▶️ How to Run
```bash
pip install -r requirements.txt
python app.py
